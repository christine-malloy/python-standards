# Python Standards

Collection of community driven standards for structuring projects and code.

## Usage

Start virtual environment
```sh
python3 -m venv venv
source venv/bin/activate
```

Install packages

```sh
./scripts/dev_install.sh
```

Run hello world app:
```sh
hello_world
```

Run webserver:
```sh
webserver
```

Run tests:
```sh
pytest
```
